About ObjectVision

ObjectVision is an advanced web application designed to perform real-time and image-based object detection using deep learning models built with PyTorch and Flask.

The platform was developed by Shreyas Hadawale as part of a full-stack ML integration project, combining computer vision, Flask web services, and frontend design using Tailwind CSS.
