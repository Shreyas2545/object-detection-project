About The Project

Methodology

Deep Learning Models

YOLO (You Only Look Once)

A state-of-the-art, real-time Object detection system that frames object detection as a regression problem, achieving remarkable speed and accuracy by processing images in a single neural network pass.

CNN (Convolutional Neural Network)

A deep learning architecture designed for processing grid-like topology data such as images, using convolutional layers to automatically extract hierarchical feature representations.

ResNet-18

An 18-layer residual neural network using skip connections to enable training of deep networks, avoiding vanishing gradients and achieving superior accuracy.

MobileNet

A lightweight deep neural network architecture optimized for embedded systems and mobile devices, designed using depthwise separable convolutions for reduced computation cost.

Machine Learning Models

KNN (K-Nearest Neighbors)

A non-parametric, instance-based learning algorithm that classifies objects based on the majority class among their k nearest neighbors in feature space.

SVM (Support Vector Machine)

A supervised learning algorithm that finds the optimal hyperplane in high-dimensional space to maximize separation between classes, effective for linear & non-linear classification.

Decision Tree

A tree-structured model predicting outcomes using hierarchical splitting of features, enabling interpretable rule-based classification.

Random Forest

An ensemble learning approach that constructs multiple decision trees and outputs aggregated predictions, improving accuracy and reducing overfitting.
